통합유틸 프로그램을 설치했는데도 바로콘구동이 제대로 않되면 배치프로그램이 제대로 수행되는지 확인해
봐야 합니다.
시작에 CMD창을 오픈하시고 ipconfig 명령을 입력 하신후 "ipconfig"은(는) 내부 또는 외부 명령,실행할수.....
배치 파일이 아닙니다. 라고 나올경우 배치명령어가 제대로 설정이 않된것입니다.
내컴퓨터=>속성=> 고급시스템설정=>고급=>환경변수=> 시스템변수중 path 값에 ";C:\Windows\System32"
추가후 다시 통합유틸 프로그램을 설치해주세요.

증상: 최초 바로콘 설치시 자동설치 와 통합설치를 통하여 정상적으로 설치가 됬으나 로그인이 않되며 TOBESOFT 폴더하위에 Component와 Cache 폴더가 생성이 않되는 경우
